# Este aplicativo é baseado no Tinder com as funcionalidades de adicionar usuários ao sistema 
# e permitir que estes atribuam 'like' ou 'dislike' para outros usuários. A adição de usuário 
# é baseada na API de usuários do Github. 
# Aplicativo baseado em NodeJS (backend) e React.JS (frontend). Utiliza também o repositório
# YARN e os plugins: Axios, Mongoose, Cors, Router-dom.
# Aplicativo desenvolvido no curso online Omnistack promovido por Rockectseat.com.br
